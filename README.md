## 🛡️ 概要（Overview）

**「エンジニアの目」を、すべてのブラウザへ。**

SecutI は、ブラウザのフロントエンドにおける不審な挙動を監視するセキュリティ拡張機能です。
外部サーバーへデータを送信することなく、ブラウザ内で完結する**サーバーレス設計**を採用しています。

### 💡 開発の動機（Philosophy）
インターネット上の詐欺・フィッシングサイトは年々巧妙化しています。見た目だけで本物と偽物を区別することは困難になりつつあります。

しかし、どれほど巧妙に偽装しても、**裏側の挙動（JavaScriptや通信）には必ず痕跡が残ります。**
通常、エンジニアは `F12` キー（開発者ツール）を開き、Network や Console を確認して危険を察知します。
**SecutI は、この「エンジニアの視点」を自動化・補助するために設計されました。**

## 🚀 主な機能（Features）

### 1. 🕵️‍♂️ 挙動監視・警告 (Behavior Analysis)
UIの見た目ではなく、JavaScriptの挙動と通信パターンを監視します。
* **フォーム送信先監視:** 入力データが表示中のドメインとは異なる場所へ送信されるのを検知します。
* **ダウンロード保護:** 不審なファイル形式や、Blob/Data URIを利用したドライブバイダウンロードをブロックします。

### 2. 📦 Modシステム (Extension Rules)
**あなたのセキュリティルールを、JSONで追加できます。**
AviUtlのプラグインのように、設定ファイル（`.json`）を読み込むことで検知ルールを拡張可能です。
* 特定の詐欺サイト用Mod
* 誤検知回避用Mod
これらをユーザー間で共有し、SecutIを自分好みに強化できます。

### 3. ⚙️ プライバシーファースト
* **完全ローカル動作:** 解析のためにURLや入力データを外部サーバーに送信することはありません。
* **Manifest V3準拠:** Chromeの最新仕様に準拠した設計です。

### 4. 📱 モバイル対応
Android版 Kiwi Browser 等で使用可能です。PC版と同等の検知ロジックがスマホでも動作します。

## 📖 使用方法


1.  **インストール**
    Chromeウェブストア、またはGitHubのリリースページから拡張機能をインストールします。
2.  **ブラウジング**
    通常通りWebサイトを閲覧してください。SecutIはバックグラウンドで監視を続けます。
3.  **警告表示**
    不審な挙動、またはModルールに一致する挙動を検知すると、画面に警告が表示されます。
4.  **Modの追加（オプション）**
    拡張機能のアイコンをクリックし、「Settings」タブから `.json` ファイルをインポートしてください。

## 🧩 現在検知している挙動（v1.0）

* フォーム送信先が表示中ドメインと異なる（Cross-Origin）
* 不自然にランダム、または使い回される送信先URL
* ユーザー入力を含む通信の即時外部送信（Keylogger挙動）
* 偽装された「私はロボットではありません」UI
* Modルールに基づくカスタム検知
